## 📝 Инструкция по запуску

### 1. Установка PostgreSQL
Скачайте и установите PostgreSQL с официального сайта:
https://www.postgresql.org/download/

### 2. Развертывание базы данных

**Windows:**
```cmd
cd currency-tracker\scripts
deploy.bat
```

**Linux/Mac:**
```bash
cd currency-tracker/scripts
chmod +x deploy.sh
./deploy.sh
```

### 3. Установка Python зависимостей
```bash
cd currency-tracker/parser
pip install -r requirements.txt
```

### 4. Настройка конфигурации
```bash
cp .env.example .env
# Отредактируйте .env файл
```

Пример .env:
```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=currency_tracker
DB_USER=postgres
DB_PASSWORD=your_password
```

### 5. Запуск парсера
```bash
python main.py
```

### 6. Использование CLI
```bash
# Анализ тренда
python cli.py trend USD/RUB 7

# Поиск арбитража
python cli.py arbitrage

# Визуализация
python cli.py visualize USD/RUB 30

# Экспорт данных
python cli.py export csv USD/RUB 30
```

### 7. Запуск API сервера
```bash
python api_server.py
```

API будет доступен по адресу: http://localhost:5000

### 8. Запуск планировщика
```bash
python scheduler.py
```
