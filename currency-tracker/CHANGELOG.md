# Changelog

Все значимые изменения проекта Currency Tracker документируются в этом файле.

## [1.1.0] - 2026-03-14

### Добавлено
- Модуль анализа данных (analyzer.py)
- REST API сервер (api_server.py)
- CLI интерфейс (cli.py)
- Планировщик задач (scheduler.py)
- Визуализация данных (visualizer.py)
- Экспорт данных (exporter.py)
- Система алертов и уведомлений
- Функции для работы с курсами:
  - Конвертация валют
  - Расчет кросс-курсов
  - Прогнозирование (SMA)
  - Определение тренда
  - Расчет RSI индикатора
  - Поиск арбитража
  - Экспорт в JSON

### Улучшено
- Расширенная аналитика
- Автоматизация через планировщик
- API для интеграции с другими системами
- Графическая визуализация
- Экспорт в CSV/JSON

## [1.0.0] - 2026-03-14

### Добавлено
- Структура базы данных PostgreSQL
- Таблицы: currencies, data_sources, exchange_rates, currency_statistics, parsing_log
- Индексы для оптимизации запросов
- Триггеры для автоматизации
- Представления для аналитики
- Парсер данных ЦБ РФ
- Парсер данных ЕЦБ
- Скрипты развертывания для Windows и Linux
- Тестовые данные
- Документация проекта
- Лицензии MIT (EN/RU)

### Возможности
- Автоматический парсинг курсов валют
- Хранение исторических данных
- Расчет статистики и волатильности
- Аналитические представления
- Логирование работы парсера
