# TODO — Currency Tracker

## Состояние проекта (14.03.2026)

### Стек

- **БД**: PostgreSQL 16, порт 5433, база `currency_tracker`
- **Python**: 3.14.3 — `C:\Users\1\AppData\Local\Python\pythoncore-3.14-64\python.exe`
- **Сервер**: FastAPI + Uvicorn, порт 8000
- **Фронтенд**: Jinja2 + Chart.js (CDN), без Node.js
- **Данные**: 7015 записей, 15.09.2025 — 14.03.2026

### Запуск проекта

```bash
# Из папки currency-tracker/parser/
python fastapi_server.py           # сервер → http://localhost:8000
python load_history.py             # перезагрузить историю 6 мес
python load_history.py 365         # загрузить год
python load_history.py --no-clear  # дополнить без очистки
python main.py                     # разовый парсинг (ЦБ РФ + ЕЦБ)
```

### Источники данных

- ЦБ РФ: `https://www.cbr.ru/scripts/XML_daily.asp` (текущий день)
- ЦБ РФ история: `https://www.cbr.ru/scripts/XML_dynamic.asp?date_req1=...&date_req2=...&VAL_NM_RQ=R01235`
- ЕЦБ: `https://www.ecb.europa.eu/stats/eurofxref/eurofxref-daily.xml`
- ЕЦБ история: `https://www.ecb.europa.eu/stats/eurofxref/eurofxref-hist.xml`

## Известные проблемы / Заметки

- ЦБ РФ публикует курс на **следующий** рабочий день после ~15:30
  - Курс на сайте cbr.ru/currency_base/daily/ = курс на завтра
  - Поэтому небольшое расхождение с текущим рыночным курсом — это нормально

- `ON CONFLICT DO UPDATE` в `database.py` — курс за текущую дату обновляется при каждом запуске парсера

- Планировщик встроен в FastAPI через `lifespan` + `threading.Thread(daemon=True)`
  - Парсинг: каждый день в 09:00
  - Проверка алертов: каждый час
  - При остановке сервера планировщик тоже останавливается

- ЕЦБ не публикует данные в выходные — пропуски в истории по субботам/воскресеньям это норма

- Для загрузки истории больше 1 года ЦБ РФ может throttle запросы — добавлен `time.sleep(0.1)` между валютами

- Flask (`api_server.py`) и FastAPI (`fastapi_server.py`) существуют параллельно
  - Flask: порт 5000 (не используется активно)
  - FastAPI: порт 8000 (основной)
  - Flask можно удалить если не нужен
