# SmartOffice

> умный офис

**Цель проекта:** учёт кадров, управление задачами, складской учёт, комплексная система.

**Целевую аудиторию:** для внутреннего использования, для HR-отдела, для всей компании.

**Масштаб:** простая база, корпоративный портал, стартап-продукт.

---

### Аббревиатуры и сокращения

Короткие, легко запоминаются, подходят для внутреннего корпоративного использования.

HRM (Human Resource Management) — классика.

EMS (Employee Management System) — система управления сотрудниками.

PMS (Personnel Management System) — система управления персоналом.

OFS (Office Staff) — офисный персонал.

SPR (Staff Personal Record) — личные записи сотрудников.

TAM (Time and Attendance Management) — управление временем и посещаемостью.

Можно добавить цифры или версию: HRM 2.0, EMS Pro.
